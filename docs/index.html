The constructor function initializes an object with red, green, blue, and alpha values.

Param Type Default Description
r number The parameter r typically represents the red component of a color in RGB (Red, Green, Blue) color model. It specifies the intensity of the red color in the range of 0 to 255.
g number The parameter g in the constructor function represents the green component of a color in RGB (Red, Green, Blue) format. It is used to define the intensity of the green color in the color representation.
b number The b parameter in the constructor function represents the blue component of a color. It is used to define the intensity of the blue color in the RGBA (Red, Green, Blue, Alpha) color model.
